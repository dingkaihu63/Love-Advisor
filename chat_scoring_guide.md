# 聊天记录七维评分详细标准

## R1 - 回复积极性（0-10分）

- 秒回/快速回复 → 8-10分
- 正常速度回复 → 5-7分
- 回复较慢但内容充实 → 4-6分
- 回复慢且简短 → 1-3分
- 经常不回 → 0分

## R2 - 话题参与度（0-10分）

- 主动开启新话题 → 8-10分
- 积极延续你的话题 → 5-7分
- 被动回应不拓展 → 3-4分
- 敷衍回应 → 1-2分
- 不回应话题 → 0分

## R3 - 情感投入度（0-10分）

- 分享私人感受/经历 → 8-10分
- 对你的分享表示关心 → 5-7分
- 礼貌但无情感波动 → 3-4分
- 冷淡疏离 → 1-2分
- 完全无情感交流 → 0分

## R4 - 主动性（0-10分）

- 经常主动找你聊天 → 8-10分
- 偶尔主动 → 5-7分
- 基本不主动但会回应 → 3-4分
- 极少主动 → 1-2分
- 从不主动 → 0分

## R5 - 特殊信号（0-10分）

- 暧昧表达/调侃/撒娇 → 8-10分
- 使用亲昵称呼/表情包 → 5-7分
- 正常友好交流 → 3-4分
- 公事公办语气 → 1-2分
- 明确保持距离 → 0分

## R6 - 一致性（0-10分）

- 热情度持续稳定 → 8-10分
- 大部分时间热情 → 5-7分
- 忽冷忽热 → 3-4分
- 逐渐变冷 → 1-2分
- 一直冷淡 → 0分

## R7 - 投资度（0-10分）

- 愿意花时间/精力为你做事 → 8-10分
- 记得你说过的小事 → 5-7分
- 有基本关心 → 3-4分
- 漠不关心 → 1-2分
- 只索取不付出 → 0分
